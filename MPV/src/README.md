# Исходники расширения

Этот каталог содержит переносимый комплект исходников для расширения 1С:Бухгалтерия 3.0.194.14.

Расширение добавляет пользовательские правила обработки банковской выписки. Правила могут:

- ограничиваться организацией, банковским счетом, направлением операции, контрагентом, ИНН и текстом назначения платежа;
- заполнять вид операции поступления/списания, контрагента, договор, банковский счет, статью ДДС, счета Дт/Кт и статью прочих доходов/расходов;
- отдельно подбирать `СчетНаОплату` для входящих оплат покупателей.

Файлы:

- `metadata/МетаданныеРасширения.md` - какие объекты нужно создать в расширении.
- `common-modules/ВП_ПравилаОбработкиВыпискиСервер.bsl` - серверная логика чтения правил, проверки условий, заполнения реквизитов, поиска счета и записи журнала.
- `extension-modules/ЗагрузкаВыпискиПоБанковскомуСчету_Расширение.bsl` - фрагмент расширения типового общего модуля.
- `catalog-forms/ВП_ПравилаОбработкиБанковскойВыписки.ФормаЭлемента.Модуль.txt` - модуль формы элемента справочника правил.
- `tests/СценарийПроверки.md` - ручной сценарий проверки на файле и DirectBank.

Ограничение текущего workspace: здесь нет выгруженного проекта расширения или информационной базы, поэтому `.cfe` не собирается автоматически. Исходники подготовлены для переноса в расширение через конфигуратор/EDT.
